Welcome to my info page for my fyp-crawler

This project aims to design and develop a Java web crawler that download several forums and extract the post contents into storage.

Tools and Libraries

Multiple tools and open source libraries that are used for the project are listed below

  • HttpComponents Client 4.5.x
  • Jsoup 1.8.3
  • MongoDB Java Driver

Crawler Design

Following are the main components of the crawler:

  • Crawler main controller: to control the crawling processes, eliminate duplicate crawling, filter the invalid/ irrelevant URLs
  • Connection manager: Using Http Components Client to handle all connection to a forum server and request execution threads to retrieve webpages from a web forum
  • Page Processor: Set of tools and methods to process with the downloaded pages to identify and extract the forum post content
  • Database: A data storage that contains all forum post contents and their attached meta-data such as username, timestamp, quote, etc.
  • Graphical User Interface (optional): to interact with user and get the input for operating the crawling process. (For example: forum to be crawl, time interval between each time the crawler check for update, etc.)
